데이터 엔지니어링이란?

데이터 엔지니어링은 원시 데이터를 수집, 변환, 저장하여 분석과 의사결정에 활용할 수 있는 형태로 만드는 과정입니다. ETL/ELT 파이프라인 구축, 데이터 웨어하우스 설계, 데이터 레이크 아키텍처 등이 주요 영역입니다.

데이터 레이크하우스(Lakehouse) 완벽 가이드

데이터 레이크와 데이터 웨어하우스의 장점을 결합한 혁신적인 아키텍처인 레이크하우스에 대해 알아봅니다.
